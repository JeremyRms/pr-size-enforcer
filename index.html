Welcome to the PR Size Enforcer GitHub App! This app helps maintain a healthy code review culture by automatically closing pull requests that exceed a configurable size threshold.

Features

  • Closes PRs exceeding a max line count (default: 500 LOC)
  • Ignores specified file types (e.g., .lock, .snap)
  • Works org-wide across all repositories
  • Configurable threshold and ignore rules

Installation

To install the app, visit the GitHub Apps page and follow the instructions.
